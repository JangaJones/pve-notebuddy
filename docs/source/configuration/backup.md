# Backup and Restore

## What to Back Up

Two different backup types exist:

- full app settings/state backup (Settings panel)
- single template export (Template Manager)

## Full State Backup

Use `Settings -> EXPORT` to download current state JSON.

Includes:

- UI preferences
- settings values
- local template catalog

Import with `Settings -> IMPORT` on another browser or machine.

## Template Backup

Use Template Manager export when you only need one template design snapshot.

## Recommended Routine

- export full settings before major changes
- keep a dated backup file naming pattern
- test restore on a second browser profile occasionally
