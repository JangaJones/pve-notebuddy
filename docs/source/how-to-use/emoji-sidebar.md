# Emoji Sidebar

The Emoji sidebar is a quick clipboard helper for row labels and status markers.

## Purpose

- provide a curated emoji set for infrastructure notes
- reduce context switching to external emoji pickers
- speed up formatting of hosts, services, warnings, and status badges

## How It Works

1. Open the Emoji tab.
2. Click any emoji chip.
3. Emoji is copied to clipboard.
4. Paste into a row icon/text field.

## Notes

- Copy uses browser clipboard APIs.
- If clipboard access is blocked by browser policy, the UI shows an inline failure hint.
