# PVE NoteBuddy Documentation

PVE NoteBuddy is a static, client-side web app for generating polished Proxmox VE note content quickly.

Use this documentation to learn:

- what the app does and how it works
- how to build notes efficiently
- how templates, settings, and backups behave
- how to self-host and contribute

## Important Security Note

Use the official project repository and official GitHub Pages deployment. Be careful with third-party forks that offer downloads or modified builds. A malicious fork can inject unsafe code into a static app.
