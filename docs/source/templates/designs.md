# Designs

Design buttons are quick layout presets for row formatting and style defaults.

## Default Design Presets

Preset buttons `1` to `5` load JSON from:

- `./templates/presets/notebuddy-template-1.json`
- `./templates/presets/notebuddy-template-2.json`
- `./templates/presets/notebuddy-template-3.json`
- `./templates/presets/notebuddy-template-4.json`
- `./templates/presets/notebuddy-template-5.json`

## Custom Design Workflow

Use a preset as starting point, adjust icon/rows/formatting, then save as local template.

## Recommendation

Keep one standard design per environment/team to improve readability across hosts.
